# Ramas-Repo
Rama: Steven-Gallo

En base al archivo .fxml diseñado por todo los integrantes de mi grupo cree una esquema base de la clase aplicationController, la cual llevara toda la logica, controles e funciones del sistema CRUD de cafeteria. 
Para eso ocupamos tableview, textfield, combobox; con esto se creo la base para que mis compañeros realizen las funcionalidades de los botoens que se implemantan en el CRUD.
Se creo el metodo de la conexion a la BD el cual llama a la logica de la clase Conexion para mostrar en el table view todo el contenido dne la BD
Y tambien se creo el metodo de mostrar alertas el cual muestra alertas tipo: INFORMATION y ERROR en base las validacions implementadas por mis compañeros.
